# DocumenterShikiDemo

[![Build Status](https://github.com/hsugawa8651/DocumenterShikiDemo.jl/actions/workflows/CI.yml/badge.svg?branch=main)](https://github.com/hsugawa8651/DocumenterShikiDemo.jl/actions/workflows/CI.yml?query=branch%3Amain)
[![Stable](https://img.shields.io/badge/docs-stable-blue.svg)](https://hsugawa8651.github.io/DocumenterShikiDemo.jl/stable/)
[![Dev](https://img.shields.io/badge/docs-dev-blue.svg)](https://hsugawa8651.github.io/DocumenterShikiDemo.jl/dev/)

This package demonstrates how to integrate [Shiki](https://shiki.style/) syntax highlighting with [Documenter.jl](https://github.com/JuliaDocs/Documenter.jl) to provide VS Code-quality highlighting for Julia documentation.

For automating this setup in new Julia packages, see [PkgTemplatesShikiPlugin.jl](https://github.com/hsugawa8651/PkgTemplatesShikiPlugin.jl).
